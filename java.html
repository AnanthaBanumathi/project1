JAVA PROGRAMMING


Course Description


  • The main ideas behind Java's Object-Oriented Programming, OOP concepts include abstraction, encapsulation, inheritance and polymorphism.

  • The Java programming language is designed for creating highly reliable software. It provides extensive compile-time checking, followed by a second level of run-time checking.

  • Keywords in Java are reserved words that have a specific meaning to the compiler and cannot be used for other purposes. They include 'class', 'public', 'static', 'void', 'if', 'else', and many more.
